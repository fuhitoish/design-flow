# 06 — モジュール A（印刷物・パンフ・同封物）

Fuhito の実務で最優先の媒体。試用案件（竹うちわ＋二つ折りポケットフォルダ）を含む **印刷物の型** を整理。

---

## 1. モジュール A で扱うもの

| 種別 | 例 |
|---|---|
| パンフレット | A4 中綴じ 8P、リーフレット |
| 封筒同封 | 既製封筒 + 同封物 |
| フォルダ | 二つ折り 2 ポケット（試用案件） |
| チラシ | 単枚 |
| カード | 名刺サイズ同封 |

---

## 2. 製本・折りの型

### 2.1 中綴じ（ステープル）

| ページ数 | 特徴 | 向く |
|---|---|---|
| 4P | 1 シート両面 | シンプル案内 |
| 8P | **定番**。表紙+裏表紙+本文 4 見開き | 宿泊・商品パンフ |
| 12P | 情報量多 | カタログ |
| 16P+ | 冊子感 | 要背表紙検討 |

**台割の型:** 表1=表紙 / 表8=裏表紙 / 2-3 見開き…（サンプル案件 Bellagio 参照）

**DPL 候補:** ✅ P0 — 8P / 12P 台割テンプレ

#### 8P 台割パターン（3 種）

印刷会社の入稿用「見開き」ではなく、**企画段階の役割割り当て**用。

**パターン A — 定番（宿泊・施設案内）**

| P# | 役割 | 備考 |
|---|---|---|
| 表1 | 表紙 | タイトル + 1 ビジュアル |
| 表8 | 裏表紙 | 連絡先・QR・ロゴ |
| 1 | オープニング | キーメッセージ |
| 2-3 | 見開き・主力訴求 | 写真大 |
| 4-5 | 詳細・商品 | 2 段組可 |
| 6 | FAQ・細部 | テキスト多め → F 視線 |
| 7 | 予約・CTA | 電話・URL |

模式図: [patterns/svg/print-8p.svg](../patterns/svg/print-8p.svg)

**パターン B — 商品カタログ寄り**

| P# | 役割 |
|---|---|
| 表1 | 表紙 |
| 1-2 | ブランドストーリー |
| 3-4 | 商品 A/B |
| 5-6 | 商品 C/D |
| 7 | 価格・CTA |
| 表8 | 裏表紙 |

**パターン C — イベント・期間限定**

| P# | 役割 |
|---|---|
| 表1 | インパクト表紙 |
| 1 | キャンペーン概要 |
| 2-3 | 特典・ギフト（うちわ等） |
| 4-5 | 利用条件 |
| 6-7 | 予約導線 |
| 表8 | 注意事項・連絡先 |

→ DPL: `PRINT-BIND-8P` + `PRINT-PAGE-ROLES-8P`（[patterns/print.json](../patterns/print.json)）

### 2.2 二つ折り（1 シート）

- A4 1 枚 → 4 面（表外・表内・裏内・裏外）
- **DPL:** P0

### 2.3 三つ折り（Z 折 / 巻き三つ折）

- 地図・手順・多言語
- パネル幅の計算が必要
- **DPL:** P1

### 2.4 二つ折り・2 ポケットフォルダ（試用案件）

```
        ┌─────────────┐
        │   表紙      │
        ├──────┬──────┤
        │ポケ1 │ポケ2 │  ← うちわ・封筒を入れる
        └──────┴──────┘
```

模式図: [patterns/svg/print-folder-2pkt.svg](../patterns/svg/print-folder-2pkt.svg)

**展開図の検討項目（実寸設計前）**

```
  [表紙面]
  ┌────────────────────────────┐
  │         FRONT COVER        │
  ├─────────────┬──────────────┤
  │  POCKET L   │  POCKET R    │  ← 折り線
  │  (glue)     │  (glue)      │
  └─────────────┴──────────────┘
       ↑ 糊しろ・折り代を別途加算
```

| 検討項目 | 内容 |
|---|---|
| 展開サイズ | 仕上がり + 糊しろ・折り代 |
| ポケット深さ | 同封物の厚み（うちわ＋封筒） |
| 視線 | **表紙で開封動機**（dontDo: 開けずに捨てる） |
| 郵送 | 規定重量・サイズ → [references/postal-japan.md](../references/postal-japan.md) |

**DPL 候補:** ✅ P0 — 2 ポケットフォルダ展開図テンプレ

## 3. ページ役割の型（台割）

---

## 4. 同封物セットの型（試用案件タイプ）

**「休暇案内＋ギフト」** パターン:

| 同封物 | デザイン対象 | メモ |
|---|---|---|
| 案内パンフ or リーフレット | 全面 | メッセージ本体 |
| 竹うちわ | **裏面のみ**（試用） | ロゴ過多に注意 |
| 封筒 | 既製 or オリジナル | サイズ連動 |
| フォルダ | 2 ポケット | 一体感のトンマナ |

**DPL 候補:** 「ギフト同封パッケージ」型（P0）

---

## 5. 入稿・仕様のチェック型

| 項目 | よくある値 |
|---|---|
| 解像度 | 350dpi（印刷会社による） |
| 塗り足し | 3mm 前後 |
| 色 | CMYK、特色指定時はパレット番号 |
| フォント | アウトライン or 埋め込み |
| うちわ | 曲面・曲面ロゴの歪み検証（実物 mock） |

**DPL:** 入稿チェックリストは `pilot/templates` に既存 → 知識としては参照リンクのみ

## 6. 郵送（DM）の制約
