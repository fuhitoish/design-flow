# 05 — 比率・プロポーション

用紙サイズ・見開き・要素の縦横比を決めるための整理。試用 FB「和の比率 1.414」に対応。

---

## 1. なぜ比率が重要か

- 用紙そのものが **√2 系**（A 判）なら、中身も同系統の比率と揃えると **違和感が少ない**
- 見開き・折り・写真トリミングを **数値で再現** できる
- 「なんとなく」から **選べる型** にするのが DPL の目的

---

## 2. 主要な比率

### 2.1 黄金比（φ ≈ 1.618）

| 項目 | 内容 |
|---|---|
| 定義 | 大:小 = (大+小):大 |
| 印象 | 調和・美・自然 |
| 用途 | 見出しサイズ階層、要素の横幅分割、キービジュアル枠 |
| **DPL** | P1 |

### 2.2 白銀比（大和比 ≈ 1.414 = √2）

| 項目 | 内容 |
|---|---|
| 定義 | 1 : √2 |
| 印象 | **和・安定・縦長の品格**（試用時に言及） |
| 用途 | 要素の縦横、余白分割、掛け軸・和室の意匠と親和 |
| 関連 | **A 系列用紙**（A4 = √2 展開）と同系統 |
| **DPL** | ✅ P0 |

### 2.3 三分割（1 : 1 : 1）

- 横・縦を 3 等分。構図の基準線
- **DPL:** P0（04 と共通）

### 2.4 ダブル黄金長方形

- 細長いバナー・背表紙的な要素
- **DPL:** P2

---

## 3. 用紙・印刷の比率

### 3.1 ISO A 判（√2）

| サイズ | 寸法（mm） | 比率 |
|---|---|---|
| A4 | 210 × 297 | 1 : √2 |
| A5 | 148 × 210 | 同上 |
| B5 | 182 × 257 | 同上系 |

**意味:** A4 を半分にすると A5。**中綴じ・折り** の設計はこの系統と整合させやすい。

### 3.2 国内よくある変形

- B4 変形、正方形（SNS・うちわ裏面の検討領域）
- 封筒・ポケットフォルダは **別比率**（06 参照）

---

## 4. 写真・画像のアスペクト比の型

| 比率 | 用途例 |
|---|---|
| 16:9 | 横長ヒーロー・Web |
| 4:3 | 汎用写真 |
| 3:2 | カメラ標準 |
| 1:1 | SNS・正方形ロゴ |
| 1:√2 | A4 全面・和の縦長ブロック |
| 2:1 | 超横長バナー |

**DPL 候補:** 画像枠アスペクト 6 種（P0）

---

## 5. 折りと比率

折り紙・封筒の展開図では **パネル幅の比率** が決まる。

| 折り | メモ |
|---|---|
| 二つ折り | 各面 1:1（A4→210×148.5 相当） |
| 三つ折り（Z / C） | 3 パネル。外パネル幅が微差（入れ子） |
| 観音折り | 3 面。見開き中央の幅設計が重要 |

**DPL 候補:** 折り別パネル比率（06 と統合）P0

---

## 6. 実務での使い方（Design Flow）

1. Step 1 で用紙・折りを確定  
2. Step 4 で **比率プリセット**（√2 / 黄金 / 三分割）を選択  
3. Step 5 でキービジュアル枠・段組幅に **同じ比率** を適用  
4. ゲートで「なぜこの比率か」を 1 文（例: 「A4 と同系の √2 で統一」）
